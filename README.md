# Server Actions Practice App

This project is a lightweight full-stack application created to experiment with:

- ✅ **Server Actions** in Next.js 14
- 🎨 **ShadCN** for modern UI components
- 🛠️ **Prisma** for backend database access

## Stack

- **Frontend**: Next.js 14, ShadCN UI
- **Backend**: Server Actions, Prisma

## Functionality

- 🔍 **Hotel List Filtering** – dynamic filters for browsing hotels
- 🔐 **User Authentication** – login & registration system

npm install
npm run dev
